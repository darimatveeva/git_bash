Открыть домашнюю директорию через терминал: cd /d/Тестирование/Курс\ по\ тестированию/ДЗ/git_bash/ (\ используется для экранирования пробелов)
Определить имя папки, в которой вы находитесь: pwd
Создать внутри этой папки каталог с именем test1: mkdir test1
Перейти в папку test1: cd test1
Создать файл 1,2 и 3 внутри каталога test1: touch 1, touch 2, touch 3
Проверить содержимое каталога test1: ls (без указания директории, так как мы находимся в ней)
Перейти в домашнюю директорию: cd ..
Создать папку test2 внутри домашней директории: mkdir test2
Удалить папку test2: rmdir test2
Удалить файл 2 из папки test1: cd test1 => rm 2
Создать папку в домашней директории test3 и добавить в нее два файла cd .. => mkdir test3 =>  cd test3 => touch file1.txt => touch file2.txt
Удалить папку test3: cd .. => rm -r test3
Создать папку test4 в домашней директории: mkdir test4
Переместить файлы 1 и 3 из папки test1 в папку test4: cd test1 => mv ./test1/1 ./test4 => mv ./test1/3 ./test4
Добавить в файл 1 три строки со словами line: cd .. => cd test4 => nano 1 (ввести "line" в три строки)
Посмотреть содержимое файла 1: cat 1
Добавьте в файл 3 три строки со словами line: nano 3 (ввести "line" в три строки)
Просмотрите содержимое двух файлов (1 и 3) сразу: cat 1 3
Используя один из редакторов замените все строки в файле 1: nano 1 (заменить "line" на "test")
